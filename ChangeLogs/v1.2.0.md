## [EN]ğŸ‰ yt-dlp GUI v1.2.0 Released
Bug fixes and feature improvements.

### New Features
- **Video Info Refresh Button**: Added a refresh button to the video info panel. You can now re-fetch the latest information without changing the URL.

### Improvements & Fixes
- **WAV Save Error Fix**: Fixed an error "Supported filetypes for thumbnail embedding are..." that occurred when saving in WAV format. Thumbnail embedding is now automatically skipped for formats that don't support it (such as WAV).
- **Video Info Display Fix**: Fixed an issue where the previous URL's video information was displayed when changing the URL. Old information is no longer shown until the new URL's information fetch is complete.
