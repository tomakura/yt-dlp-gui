## [JP]🎉 yt-dlp GUI v1.3.3 リリース
動画情報プレビューとバイナリ管理を中心に、速度と安定性を改善しました。

### 主な変更
- **高速プレビュー**: 軽量なメタデータ取得を優先し、必要な場合だけ `yt-dlp` にフォールバックするハイブリッド方式に変更しました。
- **ffmpeg 配布元の見直し**: `FFmpegBin` ベースの配布に切り替え、macOS Apple Silicon を含む各アーキテクチャで自動導入しやすくしました。

### 改善・修正
- **プレビューエラーの分類**: timeout / network / rate limit / authentication required / unsupported URL を区別して表示するようにしました。
- **誤ったエラーメッセージを修正**: timeout 時に「対応していないURL」と表示される問題を修正しました。
- **ffmpeg 未導入でもプレビュー可能**: 実ダウンロード前の情報確認がしやすくなりました。
- **時間範囲指定ダウンロード**: 開始時刻と終了時刻を指定してダウンロードできるようにしました。
- **キャンセル処理の改善**: `yt-dlp` / `ffmpeg` の子プロセスも含めて停止し、一時ファイルの取り扱いを改善しました。
- **設定とキューの安定化**: 壊れた保存設定で起動時に落ちにくくし、未処理キューは再開前に確認するようにしました。

## [EN]🎉 yt-dlp GUI v1.3.3 Released
This release improves speed and stability, especially around video preview and binary management.

### Major Changes
- **Faster preview pipeline**: Switched to a hybrid approach that uses lightweight metadata fetchers first and falls back to `yt-dlp` only when needed.
- **Updated ffmpeg distribution source**: Moved to `FFmpegBin`-based downloads so automatic installation works better across architectures, including macOS Apple Silicon.

### Improvements & Fixes
- **Typed preview errors**: Timeout, network, rate limit, authentication required, and unsupported URL are now reported separately.
- **Fixed misleading timeout errors**: Timeouts no longer appear as “unsupported URL”.
- **Preview works without ffmpeg**: You can inspect video information before installing full download binaries.
- **Time-range downloads**: Added support for downloading a selected start/end section.
- **Improved cancellation**: Stops `yt-dlp` / `ffmpeg` child processes and handles temporary files more reliably.
- **More robust settings and queue restore**: Prevents crashes from corrupted saved settings and asks before resuming pending queue items.
