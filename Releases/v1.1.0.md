## [EN]ğŸ‰ yt-dlp GUI v1.1.0 Released
Added useful new features and made several improvements.

### New Features
- **Batch Import**: You can now drag and drop text files (.txt) to import multiple URLs at once.
- **Automatic Folder Creation**: Automatically creates subfolders for playlists and batch downloads to keep files organized.
- **UI Improvements**:
  - Added "Video" and "Playlist" badges to the video preview header.
  - Moved playlist progress (e.g., 1 / 10) to the header for better visibility.
  - Fixed the history panel to the bottom of the screen to improve scrolling behavior.

### Improvements & Fixes
- **Playlist Resolution**: Fixed an issue where "Best Quality" information was not correctly retrieved/displayed for videos in playlists.
- **History Panel**: Fixed the layout so the history panel doesn't shift when scrolling the main screen.
- **Translations**: Added missing Japanese translations.
- **Other**: Internal code optimizations and bug fixes.
